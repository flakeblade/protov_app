# Troubleshooting

If the device is not detected, verify the USB-C cable, PD source capability, and browser permissions.

For limit faults, disable output and inspect wiring before re-enabling a channel.

## Device not detected

Try another cable, reload the lab page, and confirm the board powers up with the status LED active.
