# Getting Started

Place ProtoV MINI on your breadboard rails and connect a USB-C cable to a PD-capable supply.

After power is applied, open the lab interface to confirm both channels are detected.

## First connection

Verify rail orientation, then enable one channel at a low voltage to validate wiring before full load testing.

```bash
# Example placeholder command
protov-cli devices list
```
