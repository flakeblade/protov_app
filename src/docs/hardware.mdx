# Hardware

ProtoV MINI uses dual 2×5 pin headers for breadboard power rails and a USB-C connector for input power.

The board footprint is credit-card sized for portable bench and field use.

## Mechanical summary

Overall dimensions and header spacing are designed for standard 2.54 mm breadboard compatibility.
